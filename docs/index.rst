
Jammy (Jam)
===========

**Jammy** is a personal toolkit.

Etymology
^^^^^^^^^


* The naming is inspired from `Jyutping <https://en.wikipedia.org/wiki/Jyutping>`_ of `Qin <https://en.wiktionary.org/wiki/%E6%AC%BD>`_.



MICS
----
